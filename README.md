# 🤖 ИИ-Продюсер — Умный Telegram-бот для контента

> **Live Demo:** [Протестировать бота в Telegram](https://t.me/mind_body_script_bot))

Этот репозиторий является технической витриной коммерческого проекта. Исходный код закрыт, здесь представлена архитектура и описание реализованного функционала.

## 🎯 Какую бизнес-задачу решает бот?
Бот выступает в роли автоматизированного ИИ-продюсера. Он переводит сложные технические тексты на язык бизнеса, делает умный рерайт, генерирует контент-планы и адаптирует посты для B2B-сегмента. 
*Инструмент разработан для экономии времени при создании экспертного контента без потери качества и структуры.*

## ⚙️ Архитектура и стек технологий
Проект спроектирован с упором на отказоустойчивость, чистый код и способность держать высокие нагрузки.

* **Язык и ядро:** `Python 3.12` + `asyncio` — полная асинхронность для обработки сотен одновременных запросов без блокировки Event Loop.
* **Фреймворк:** `aiogram 3.x` — современный роутинг, использование мидлварей (middlewares) для фильтрации апдейтов.
* **База данных и кэш:** `Redis` — используется для Finite State Machine (FSM). Надежно кэширует состояния пользователей, позволяя боту «помнить» контекст многошагового диалога.

## 📂 Структура проекта
Код разнесен на независимые модули по принципам чистой архитектуры, что позволяет легко масштабировать проект.

```text
├── bot/
│   ├── handlers/      # Обработчики команд и пользовательского ввода
│   ├── middlewares/   # Перехватчики (антиспам, логирование)
│   ├── services/      # Бизнес-логика (интеграция с внешними API)
│   ├── states/        # FSM состояния (Redis)
│   └── utils/         # Вспомогательные функции и форматеры
├── .env.example       # Шаблон переменных окружения
├── requirements.txt   # Зависимости проекта
└── main.py            # Точка входа, инициализация бота и диспетчера
